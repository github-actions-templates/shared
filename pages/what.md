# Features

- 📝 **主流操作系统** - 适用于 Linux、macOS、Windows、ARM 和容器；直接在虚拟机或容器内运行。
- 🎨 **矩阵构建** - 通过矩阵工作流程可以同时跨多个操作系统和运行时版本进行测试，从而节省时间。
- 🧑‍💻 **任何语言** - 支持 Node.js、Python、Java、Ruby、PHP、Go、Rust、.NET 等。
